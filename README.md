# Dungeon-game-with-unity
The project used unity to create a simple 2d dungeon game

1.项目介绍
============================================================
我们小组将游戏设计分为了几个部分，有背景生成与人物移动，攻击设计，怪兽生成还有初始页面的设计。
不同的小组成员负责不同的部分，在最后进行合并。

2.分支设计
============================================================
我们在不同的分支中进行各自的活动，其中main为week1的部分background效果。
